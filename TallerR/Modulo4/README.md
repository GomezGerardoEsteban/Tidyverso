## Módulo 4: Inferencia Estadística

Este módulo introduce los fundamentos de la inferencia estadística, una herramienta esencial para obtener conclusiones sobre poblaciones completas a partir de información muestral. El módulo se centrará en explicar cómo, a través de muestras aleatorias, podemos estimar parámetros poblacionales y evaluar la confiabilidad de nuestras estimaciones. Aprenderás a calcular e interpretar intervalos de confianza, que proporcionan un rango de valores dentro del cual es probable que se encuentre el verdadero parámetro poblacional (como el ingreso promedio de un estado) con un determinado nivel de confianza, generalmente del 95%.

La segunda parte del módulo abordará las pruebas de hipótesis, permitiéndote contrastar afirmaciones sobre los parámetros poblacionales (por ejemplo, si el ingreso promedio de una ciudad capital es significativamente mayor al de otras ciudades). Si bien la teoría estadística subyacente es extensa, el módulo se enfocará en la aplicación práctica utilizando R, enseñándote a ejecutar e interpretar los resultados de estas pruebas. El objetivo es equiparte con las herramientas para realizar inferencias rigurosas a partir de datos muestrales, como los provenientes de encuestas, fundamentales para la investigación en ciencias sociales y la toma de decisiones basada en evidencia.
