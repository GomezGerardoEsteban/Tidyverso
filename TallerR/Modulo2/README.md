## Módulo 2: Análisis estadístico univariado

Este módulo se centra en el análisis estadístico univariado, es decir, en las técnicas para describir y resumir la información contenida en una variable a la vez. Se abordarán las herramientas fundamentales para caracterizar distribuciones, comenzando con el análisis de variables categóricas mediante el cálculo de frecuencias absolutas y relativas. Aprenderás a transformar respuestas de encuestas en categorías manejables, agrupar datos para obtener conteos y proporciones, y presentar estos resultados de manera efectiva en tablas y gráficos de barras. Además, el módulo incluye una introducción práctica a la visualización de datos geográficos, enseñándote a crear mapas temáticos que representen proporciones por país utilizando librerías especializadas como `giscoR`.

La segunda parte del módulo está dedicada al análisis de variables cuantitativas, cubriendo tanto medidas de posición como de dispersión. Aprenderás a calcular e interpretar la media, la mediana y la moda, comprendiendo cuándo es más apropiado utilizar cada una según las características de los datos. Posteriormente, el módulo se enfoca en las medidas de dispersión, incluyendo la varianza, la desviación estándar y el rango, que permiten evaluar la variabilidad de las observaciones alrededor de los valores centrales. Finalmente, explorarás métodos para visualizar estas distribuciones mediante histogramas, gráficos de cajas (boxplots) y gráficos de línea para analizar tendencias temporales, proporcionándote un conjunto completo de herramientas para el análisis exploratorio de datos.
