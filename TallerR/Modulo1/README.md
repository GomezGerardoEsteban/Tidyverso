## Módulo 1: Introducción al IDE RStudio y Manipulación de bases de datos

En este módulo introductorio se abordarán los fundamentos necesarios para comenzar a utilizar R y RStudio de manera efectiva en el análisis de datos. La clase se centrará en dos áreas principales: la familiarización con el entorno de trabajo y las herramientas básicas para la manipulación de información. En primer lugar, se explorará el entorno integrado de desarrollo RStudio, explicando la función de sus cuatro cuadrantes principales **(editor de código, consola, entorno y visualizador)** y la importancia de configurar correctamente el directorio de trabajo, incluyendo la creación y gestión de proyectos mediante RProjects. También se cubrirá la instalación y activación de paquetes esenciales como `tidyverse`, que facilitan y potencian las capacidades analíticas del lenguaje.

Posteriormente, el módulo se dedicará a las técnicas fundamentales de manipulación de bases de datos. Esto incluye la carga de archivos en diversos formatos (.txt, .csv, .dta, .xlsx, .RData), la exploración inicial de conjuntos de datos con comandos como `glimpse()`, `summary()`, `head()` y `View()`, y la comprensión de los diferentes tipos de variables *(numéricas, categóricas, texto y lógicas)*. Se enseñará a crear, filtrar y seleccionar variables y observaciones utilizando tanto la sintaxis base de R como las funciones del paquete `dplyr` (`mutate()`, `filter()`, `select()`), destacando el uso del operador pipe (`%>%`) para encadenar operaciones de manera legible. Finalmente, se introducirán conceptos clave como la agrupación y resumen de datos con `group_by()` y `summarise()`, la transformación entre formatos largo y ancho de los datos, y las técnicas básicas para unir diferentes bases de datos mediante `left_join()`.
