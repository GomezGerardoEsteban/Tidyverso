## Módulo 3: Análisis estadístico bivariado

Este módulo se centra en el análisis estadístico bivariado, explorando las técnicas para evaluar y cuantificar la relación entre dos variables. Aprenderás a utilizar medidas fundamentales como la covarianza, que indica la dirección de la relación (positiva o negativa) entre variables continuas, y la correlación, que además de la dirección mide la intensidad de dicha relación en una escala de -1 a 1. El módulo profundiza en la interpretación práctica de estos estadísticos, incluyendo la realización de pruebas de hipótesis (`cor.test`) para determinar la significancia estadística de las asociaciones encontradas, utilizando ejemplos reales como la relación entre el desarrollo económico (PIB per cápita) y los valores democráticos.

La segunda parte del módulo introduce la regresión lineal simple, una herramienta poderosa para modelar relaciones causales y predecir el comportamiento de una variable en función de otra. Aprenderás a estimar e interpretar los coeficientes de regresión utilizando la función `lm()`, comprendiendo cómo cambios en la variable independiente afectan a la variable dependiente. Además, el módulo aborda la importancia de las transformaciones de datos, como el uso de logaritmos, para mejorar el ajuste del modelo cuando las variables no cumplen supuestos de normalidad, culminando con la visualización de estas relaciones mediante gráficos de dispersión con rectas de regresión.
