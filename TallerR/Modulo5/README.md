## Módulo 5: Visualización de datos con ggplot2

Este módulo se enfoca en la visualización avanzada de datos utilizando el paquete `ggplot2`, transformando análisis estadísticos en representaciones gráficas comunicativas y de alta calidad. Aprenderás a crear visualizaciones complejas y personalizadas, comenzando con la elaboración de mapas temáticos que representan variables continuas, como el índice de democracia por país. El módulo guía a través del proceso de definir paletas de colores personalizadas, establecer intervalos de clasificación y aplicar estilos detallados para producir mapas informativos y visualmente atractivos, incluyendo elementos como rosas de los vientos y ajustes tipográficos.

La segunda parte del módulo explora la creación de gráficos de dispersión enriquecidos para analizar relaciones bivariadas, como la correlación entre el PIB per cápita y los niveles de democracia. Aprenderás a personalizar múltiples atributos gráficos (forma, color, tamaño de puntos) en función de variables adicionales, añadir rectas de regresión, implementar etiquetas inteligentes que eviten solapamientos y diseñar leyendas personalizadas. Finalmente, el módulo enseña a combinar múltiples gráficos (como un mapa y un diagrama de dispersión) en una sola visualización cohesiva utilizando el paquete `patchwork`, culminando con la exportación de los gráficos en alta resolución para su uso en publicaciones o informes.
